## Comandos de Git
- Verificar el estado (status) de un repositorio local. (Git status)

- Agregar archivos individualmente o de forma global. (git add <nombre_archivo>)

- Agregar comentarios al commit (confirmación). (git commit -m "Tu mensaje aquí")

- Subir tus cambios al repositorio remoto. (git push origin <rama>)

- Crear, navegar y eliminar ramas (branches). Crear: (git branch <nombre_rama>), Navegar: (git checkout <nombre_rama>), Eliminar: (git branch -d <nombre_rama>)

- Revertir un repositorio a un commit específico. (git reset --hard <hash_commit>)
